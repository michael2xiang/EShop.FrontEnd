# 电商店前端
将一步一步搭建电商网站显示信息的架构，主要用于学习、交流。
参考书籍：《Asp.net设计模式》

目前主要添加了：

 - EShop.Storefront.Core： 基础设施模块，应用依赖于接口，提高灵活性；包含Unit Of Work事务模式；
 - EShop.Storefront.Respository.NHibernate：模型与数据库映射
 - EShop.Storefront.Respository.Model：模型
 - EShop.Storefront.Respository.Servier:     服务层基于Reponse-Reply模式；模型与模式视图（ModelView）,使用AutoMapper映射。
 - EShop.Storefront.Respository.UI.Web.MVC 页面显示
